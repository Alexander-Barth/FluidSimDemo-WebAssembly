Layered ocean model implemented in Julia and compiled to WASM.

This simulation show a vertical section of a layered ocean model where every layer has the density indicated by the density profile to the left. The density must increase with depth. Mode 1 is the barotropic mode, mode 2, the first baroclinic mode and so on. By changing the initial perturbation, the model is restarted.
